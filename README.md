# mash
My own programming language made from scratch.

# reminders
OK, don't expect too much from this. It is a stack-based programming language and it is built in C. You use my own extenstion for the files called '.mash'.

# install guide
So you can compile by running 'clang mash.c -o mash' or if you aren't in the directory, 'clang /path/to/mash.c -o /path/to/mash'. Obviously also replace /path/to/mash.c and /path/to/mash with the ACTUAL path to mash.c or mash. So yeah. :)

#
Oh yeah, one more thing. If you want to run a file, you do './mash file.mash'. Do this while you're in your mash directory and when you've compiled the Unix Executable File (or the UEF). IT MUST BE .MASH. ANY OTHER EXTENSTION PROBABLY WON'T WORK.

#
A Unix system is heavily, heavily recommended unless you've got WSL or some weird compatibility layer shit.
